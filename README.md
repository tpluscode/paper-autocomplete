# \<paper-autocomplete>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i paper-autocomplete
```

## Usage
```html
<script type="module">
  import 'paper-autocomplete/paper-autocomplete.js';
</script>

<paper-autocomplete></paper-autocomplete>
```
